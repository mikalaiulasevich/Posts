## Edge cases
- [ ] API списка недоступен на первом запуске.
- [ ] API details недоступен.
- [ ] Storage пустой или повреждённый.
- [ ] Быстрое повторное нажатие toggle favorite.
- [ ] Переход в details для поста, который уже есть в списке, но details ещё не загружены.
